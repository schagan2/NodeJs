Welcome to UNCC Runners Website

The atheletics are encouraged to follow their passion. This website provides the list of running events conducted. One who is interested can create an account for free and be able to participate in the events.
An event details such as event date, event area and the sponsors for the event are clearly mentioned. Along with this the benefits of doing the particular run. One can choose any type of run and can participate. All the interested events are displayed in the event saved connection page where one can update or delete an event of their wish.
